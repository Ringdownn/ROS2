\begin{abstract}
This project presents \textbf{Catch Turtle All}, a ROS2-based robotic simulation system built on the \texttt{turtlesim} environment. The system demonstrates autonomous target pursuit, action-based task execution, and chain formation following through five cooperating ROS2 nodes. A master turtle autonomously discovers and pursues dynamically spawned target turtles, while caught turtles form a snake-like chain behind the master. The project showcases key ROS2 concepts including custom Action interfaces, topic-based state sharing, service-based spawning, and robust system design.
\end{abstract}
